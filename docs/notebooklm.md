# NotebookLM 실습 가이드 — 내 자료 기반 AI 활용

**NotebookLM**은 Google이 만든 *내 자료 기반 AI 도구*입니다.
내가 올린 문서·웹·영상을 바탕으로 질문에 답하고, 요약·오디오·슬라이드 같은
다양한 자료를 자동으로 만들어 줍니다.

---

## 1. 소개

### NotebookLM이란

NotebookLM은 ChatGPT 같은 일반 챗봇과 달리, **내가 직접 올린 자료(출처)** 안에서
답을 찾습니다. 그래서 강의 자료·논문·보고서를 올려 두고 "이 자료들"에 대해
질문하고 정리하는 데 특히 좋습니다.

### RAG 방식 — 왜 할루시네이션(환각)이 적은가

- 일반 챗봇은 모델이 학습한 지식으로 답을 **생성**하다 보니, 사실이 아닌 내용을
  그럴듯하게 지어내는 **할루시네이션(환각)** 이 생길 수 있습니다.
- NotebookLM은 **RAG(검색 증강 생성, Retrieval-Augmented Generation)** 방식입니다.
  먼저 내 출처에서 관련 내용을 **찾고(retrieval)**, 그 내용에 **근거해서 답**합니다.
- 답변에는 **인용(citation) 번호**가 붙어, 어느 문장이 근거인지 바로 확인할 수 있습니다.
- 출처에 없는 내용은 보통 지어내지 않고 "정보가 없다"고 답합니다. → 5장에서 직접 확인합니다.

| 구분 | 일반 챗봇 | NotebookLM |
|---|---|---|
| 답의 근거 | 모델이 학습한 지식 전체 | **내가 올린 출처** |
| 인용 표시 | 보통 없음 | **있음** (근거 문장으로 이동) |
| 할루시네이션 | 상대적으로 많음 | 상대적으로 **적음** |
| 잘 맞는 일 | 폭넓은 일반 질문 | **특정 자료 분석·정리** |

## 3. 시작하기

1. 브라우저에서 [notebooklm.google.com](https://notebooklm.google.com) 에 접속합니다.
2. **Google 계정으로 로그인**합니다.
3. **`새 노트북`**(Create / New notebook) 버튼을 누릅니다.
4. 출처(소스)를 추가하는 화면이 나타납니다. → 4장으로 이어집니다.

> 💡 노트북(Notebook)은 *하나의 주제를 위한 작업 공간*입니다.
> 주제별로 노트북을 따로 만들면 자료가 섞이지 않습니다.

---

## 4. 출처(소스) 추가하기 — 종류별 안내

NotebookLM은 아래 **4가지 방식**으로 출처를 추가할 수 있습니다.

### ① 파일 첨부

| 종류 | 형식 예시 |
|---|---|
| 문서 | PDF (`.pdf`) |
| 텍스트 | 일반 텍스트(`.txt`), 마크다운(`.md`) |
| Google | Google 문서, Google 슬라이드 |
| 이미지 | PNG, JPG 등 |
| 오디오 | MP3 등 음성 파일 |

### ② 웹페이지 URL

- 공개된 웹페이지 주소를 붙여넣으면 그 페이지 내용이 출처로 들어옵니다.
- ⚠️ 로그인이 필요한 페이지, 일부 동적 페이지는 내용이 제대로 안 들어올 수 있습니다.

### ③ YouTube 동영상

- 공개된 YouTube 영상의 URL을 붙여넣습니다.
- NotebookLM은 영상의 **자막(스크립트)** 을 읽습니다 → **자막이 있는 영상**이 잘 됩니다.
- ⚠️ 비공개 영상, 자막이 없는 영상은 제한됩니다.

### ④ 텍스트 직접 붙여넣기

- 복사한 글을 그대로 붙여넣어 출처로 추가합니다.

### 💡 잘 업로드되는 자료로 준비하는 팁

- **PDF는 "글자가 선택되는" 파일**로 준비하세요. 스캔한 이미지 PDF는 내용 인식이
  약합니다. 가능하면 텍스트 PDF를, 스캔본은 OCR(문자 인식) 처리된 것을 사용하세요.
- 표·그림이 많은 문서는, 핵심 내용을 정리한 텍스트를 함께 올리면 더 잘 인식됩니다.
- 너무 큰 파일이나 긴 영상은 **나눠서** 올리세요. 출처별 분량과 노트북당 출처 개수에
  한도가 있습니다. (무료 기준 노트북당 약 50개 — 한도는 변경될 수 있습니다.)
- 웹페이지가 잘 안 들어오면, 그 내용을 복사해 **④ 텍스트 붙여넣기**로 추가하세요.
- 자료를 추가한 뒤 잠깐 기다리면 분석이 끝납니다. 그 후부터 질문할 수 있습니다.

## 5. 실습 1 — 출처에 질문하기 (RAG·할루시네이션 비교)

### 목표
NotebookLM이 **출처에 근거해서** 답하고, 출처 밖 내용은 **지어내지 않는다**는 것을
직접 눈으로 확인합니다.

### 준비
4장에서 출처를 1개 이상 추가한 노트북

### 진행 단계 — 이 프롬프트를 그대로 따라 해 보세요

| 단계 | 예시 프롬프트 | 확인 포인트 |
|---|---|---|
| ① 출처에 **있는** 내용 질문 | "이 자료의 핵심 주장 3가지를 **인용과 함께** 알려줘" | 답변 옆 **인용 번호**를 클릭 → 실제 근거 문장으로 이동하는지 |
| ② 구체적 수치 확인 | "○○에 대한 연도·수치를 출처에서 찾아 인용해 알려줘" | 인용된 원문과 답의 숫자가 일치하는지 |
| ③ 출처에 **없는** 내용 질문 | "이 자료에 [자료에 없는 주제]에 대한 내용이 있어? 있으면 인용해 보여줘" | **지어내지 않고** "출처에 없다"고 답하는지 |
| ④ 일반 챗봇과 비교 | 같은 ③ 질문을 ChatGPT·Gemini 등에 입력 | 출처 없는 도구는 그럴듯하게 **지어낼 수 있음** → 차이를 체감 |

### 결과 확인
- 인용 번호를 누르면 **근거가 된 원문**으로 이동합니다.
- 출처에 없는 내용을 물으면, NotebookLM은 보통 **답을 만들어내지 않습니다.**
- ④번 비교를 통해 "출처 기반 RAG"가 왜 더 신뢰할 만한지 직접 느낄 수 있습니다.

> 💡 더 엄격하게 확인하려면, 질문 끝에 이 문장을 덧붙이세요.
> *"출처에 근거가 없으면 답하지 말고 '출처에 없음'이라고만 해줘."*

> ⚠️ NotebookLM도 100% 완벽하지는 않습니다. **인용을 직접 눌러 근거를 확인**하는
> 습관이 가장 안전합니다.

## 6. 생성 기능 한눈에 보기

NotebookLM은 출처를 바탕으로 다양한 자료를 자동 생성합니다.
(보통 화면 오른쪽의 **Studio** 패널에서 만듭니다.)

| 기능 | 무엇을 만드나 | 언제 쓰면 좋은가 |
|---|---|---|
| 요약 · 브리핑 문서 | 핵심을 정리한 문서 | 자료를 빠르게 파악할 때 |
| 스터디 가이드 | 학습용 정리 + 연습 문제 | 공부·시험 대비 |
| FAQ | 자주 묻는 질문과 답 | 자료의 핵심을 Q&A로 |
| 타임라인 | 시간 순 정리 | 사건·역사 자료 |
| 마인드맵 | 개념 구조를 그림으로 | 자료 전체 구조 파악 |
| 오디오 개요 | AI 진행자 2명의 대화 팟캐스트 | 들으면서 학습 |
| 동영상 개요 | 설명 동영상 | 시각 + 음성 요약 |
| 인포그래픽 | 한 장짜리 시각 요약 이미지 | 핵심을 시각화·공유 |
| 슬라이드 | 발표용 슬라이드 | 발표 자료 초안 |
| 플래시카드 | 암기용 카드 | 용어·개념 암기 |
| 퀴즈 | 문제 세트 | 이해도 점검 |

> ⚠️ 생성 기능의 이름·위치는 자주 바뀝니다. 정확한 이름이 안 보이면
> **Studio 패널**에서 비슷한 항목을 찾아보세요.

## 7. 실습 2 — 학습 자료 만들기

### 목표
출처를 바탕으로 요약·스터디 가이드·FAQ 같은 학습 자료를 자동 생성합니다.

### 진행 단계
1. 화면 오른쪽 **Studio** 패널을 엽니다.
2. **`스터디 가이드`** 를 선택해 생성합니다.
3. 생성된 내용을 확인하고, 필요하면 **노트(메모)로 저장**합니다.
4. 같은 방식으로 **FAQ**, **브리핑 문서**, **타임라인**도 만들어 봅니다.

### 결과 확인
생성된 자료의 내용이 출처와 맞는지, 인용이 제대로 달려 있는지 확인합니다.

---

## 8. 실습 3 — 오디오 개요(한국어 팟캐스트) 만들기

### 목표
내 자료를 AI 진행자 2명이 대화로 풀어 주는 **오디오(팟캐스트)** 를 만듭니다.

### 진행 단계
1. **출력 언어를 한국어로 설정**합니다. (설정에서 언어를 한국어로 변경)
2. **Studio** 패널에서 **`오디오 개요`** 를 생성합니다.
3. (선택) **맞춤설정**에서 다룰 주제나 청중 수준을 지정합니다.
4. 생성에는 몇 분이 걸립니다 — 잠시 기다립니다.
5. 완성되면 **재생 / 다운로드 / 공유**할 수 있습니다.

> 💡 오디오 개요는 기본 언어가 영어인 경우가 많습니다. **한국어로 들으려면
> 먼저 출력 언어를 한국어로 바꾼 뒤** 생성하세요.

---

## 9. 실습 4 — 인포그래픽 · 슬라이드 만들기

### 목표
자료를 한눈에 보이는 **시각 자료**로 바꿉니다.

### 진행 단계
1. **Studio** 패널에서 **`인포그래픽`** 을 생성합니다 → 한 장짜리 요약 이미지가 만들어집니다.
2. **Studio** 패널에서 **`슬라이드`** 를 생성합니다 → 발표용 슬라이드가 만들어집니다.
3. (선택) 맞춤설정에서 강조할 주제나 범위를 지정합니다.
4. 결과물을 **다운로드**해 발표·공유에 활용합니다.

> ⚠️ 자동 생성된 인포그래픽·슬라이드는 **초안**입니다. 발표 전에 내용과 디자인을
> 반드시 검토하고 다듬으세요.

---

## 10. 공유 및 협업

- **노트북 공유**: 공유 버튼을 눌러 특정 사람을 초대하거나 링크로 공유합니다.
- **권한 설정**: 보기 전용 / 편집 가능 권한을 정할 수 있습니다.
- **활용 예**:
  - 강의 자료 노트북을 만들어 **학생들과 공유** → 학생이 직접 질문하며 학습
  - 팀 자료 노트북을 **공동 작업** → 함께 출처를 모으고 정리

## 11. 자주 묻는 점 · 주의사항

### 문제 해결

| 증상 | 원인 | 해결 |
|---|---|---|
| PDF 내용이 인식되지 않음 | 스캔한 이미지 PDF | 글자가 선택되는 텍스트 PDF / OCR 처리된 자료 사용 |
| YouTube 영상이 안 올라감 | 자막 없음 / 비공개 영상 | 자막이 있는 공개 영상 사용 |
| 웹페이지 내용이 비어 있음 | 로그인 필요 / 동적 페이지 | 내용을 복사해 텍스트로 추가 (4장 ④) |
| 출처를 더 못 올림 | 노트북당 출처 한도 도달 | 새 노트북을 만들거나 자료를 정리 |
| 오디오 개요가 한국어가 아님 | 출력 언어 설정 | 언어를 한국어로 바꾼 뒤 다시 생성 (8장) |
| 답이 부정확함 | 출처가 부족하거나 질문이 모호 | 인용 확인, 출처 보강, 질문을 더 구체적으로 |

### 주의사항

1. **인용을 직접 눌러 근거를 확인**하세요 — RAG도 완벽하지는 않습니다.
2. **저작권·민감정보**가 있는 자료는 업로드·공유에 주의하세요.
3. 자동 생성물(요약·슬라이드·오디오 등)은 **초안**입니다 — 사람이 검토·수정해야 합니다.
4. NotebookLM의 기능·한도·메뉴 이름은 **수시로 바뀝니다.** 화면이 다르면 비슷한 항목을 찾아보세요.
