# Claude Cowork 활용 가이드 — 협업과 업무 자동화

Claude를 단순 질문·답변을 넘어 **팀 협업과 실제 업무 자동화**에 활용하는 방법을 안내합니다.
**claude.ai 웹**과 **Claude Desktop 앱**을 함께 사용합니다.

> 💡 이 문서는 코드 작성 도구인 **Claude Code**와는 다릅니다.
> (Claude Code는 [vscode-claude.md](vscode-claude.md) 참고)
> 여기서는 **파일·문서·데이터 작업**을 Claude에게 맡기는 방법을 다룹니다.

## 1. 소개

### Claude Cowork란

**Claude Cowork**는 Claude를 *혼자 쓰는 챗봇*이 아니라, **팀과 함께 일하는 업무 도구**로
활용하는 것을 말합니다.

- **팀 협업**: claude.ai 팀 플랜에서 팀원과 대화·프로젝트(Project)를 공유하며 함께 작업
- **업무 자동화**: 커넥터(MCP)·스킬을 붙이면 파일 정리, 웹 정보 수집, 문서 생성 같은
  실제 업무를 Claude에게 맡길 수 있음

### 준비물

| 항목 | 설명 |
|---|---|
| Claude 계정 | [claude.ai](https://claude.ai) 가입. 팀 협업(7장)에는 **팀 플랜** 권장 |
| Claude Desktop 앱 | 내 PC 파일을 다루는 예제(4장)에 필요 — 2장에서 설치 |
| 인터넷 연결 | AI 응답·웹 기능에 필수 |
| (예제 1용) Node.js | 파일 커넥터 종류에 따라 필요할 수 있음 — 루트 [README](../README.md) "개발 환경 준비" 참고 |

## 2. Claude Desktop 앱 설치

내 PC의 파일을 직접 다루는 예제(4장)에는 **Claude Desktop 앱**이 필요합니다.
(웹만 쓰는 예제 2·3은 앱 없이 [claude.ai](https://claude.ai) 에서도 가능합니다.)

1. **다운로드**: [claude.ai/download](https://claude.ai/download) 에 접속해 **Windows용** 설치 파일을 받습니다.
2. **설치**: 받은 파일을 실행하고 안내에 따라 설치합니다. (대부분 자동으로 진행됩니다.)
3. **로그인**: 앱을 실행하고 Claude 계정으로 로그인합니다.
4. **확인**: 대화 화면이 뜨면 설치 완료입니다.

> 💡 claude.ai 웹과 **같은 계정**으로 로그인하면, 웹과 앱의 대화·프로젝트가 연동됩니다.

## 3. 확장 기능 이해하기 — MCP · Skills · 플러그인

Claude는 기본적으로 *대화*만 합니다. 여기에 **기능을 붙이면** 외부 도구·데이터를
다룰 수 있게 됩니다. 자주 나오는 세 가지 용어를 정리합니다.

### MCP (커넥터)

- **MCP**(Model Context Protocol)는 Claude를 **외부 도구·데이터와 연결**하는 표준 규격입니다.
- claude.ai·Claude Desktop에서는 보통 **"커넥터(Connector)"** 라는 이름으로 나타납니다.
- 연결 예: 내 PC 폴더, Google Drive, 웹, 사내 데이터베이스 등
- 켜는 곳: **설정(Settings) → 커넥터(Connectors)**

### Skills (스킬)

- **스킬**은 *특정 작업을 잘 해내는 능력 묶음*(작업 지침 + 참고 자료)입니다.
- 예: 엑셀 파일 만들기, PowerPoint 만들기, 정해진 형식의 문서 작성
- 사용자가 일일이 부르지 않아도, Claude가 작업에 맞는 스킬을 **알아서 골라 사용**합니다.
- 켜는 곳: **설정 → 기능(Features)/스킬(Skills)**

### 플러그인

- **플러그인**은 *스킬·명령·커넥터 여러 개를 하나로 묶은 꾸러미*입니다.
- 한 번 설치로 관련 기능 세트를 통째로 추가할 수 있습니다.
- 주로 **Claude Code**에서 쓰이는 개념이며, claude.ai·Desktop에서는 보통
  커넥터와 스킬을 **직접 하나씩 켜서** 사용합니다.

### 한눈에 비교

| 구분 | 한 줄 설명 | 비유 |
|---|---|---|
| **MCP (커넥터)** | Claude를 외부와 연결하는 통로 | 콘센트·케이블 |
| **스킬** | 특정 작업을 해내는 능력 | 전문 기술 |
| **플러그인** | 여러 기능을 묶은 꾸러미 | 도구 세트 |

> ⚠️ 메뉴 이름(커넥터 / 확장 / 기능 등)과 위치는 앱·웹 **버전에 따라 조금씩 다릅니다.**
> 정확한 이름이 안 보이면 **설정** 화면에서 비슷한 항목을 찾아보세요.

## 4. 예제 1 — 특정 폴더의 파일 다루기

> **사용 환경: Claude Desktop 앱** (claude.ai 웹은 내 PC 폴더에 접근할 수 없습니다.)

### 목표
내 PC의 특정 폴더 안 파일들을 Claude가 직접 보고, 검색·요약·정리하게 합니다.

### 준비
1. Claude Desktop 앱 설치 (2장)
2. **파일시스템 커넥터**(filesystem MCP) 추가
   - **설정 → 커넥터**에서 파일 접근 커넥터를 추가합니다.
   - **접근을 허용할 폴더**를 지정합니다. (지정한 폴더만 Claude가 볼 수 있음)
   - 커넥터 종류에 따라 Node.js 설치가 필요할 수 있습니다. (루트 README 참고)

### 진행 단계
1. 파일 커넥터가 켜져 있는지 확인합니다.
2. 대화창에 폴더를 다루는 요청을 입력합니다.
   - "`C:\작업\사진` 폴더에 어떤 파일들이 있는지 목록으로 보여줘"
   - "이름에 날짜가 없는 파일만 찾아줘"
   - "`.txt` 파일들의 내용을 한 줄씩 요약해줘"
3. 정리 작업을 맡깁니다.
   - "사진 파일을 촬영 연도별 폴더로 나눠 정리해줘"
   - Claude가 변경을 제안하면 **검토 후 허용**합니다.

### 결과 확인
지정한 폴더를 직접 열어, 요청한 대로 정리되었는지 확인합니다.

> ⚠️ Claude가 파일을 바꾸기 전에 보통 확인을 요청합니다. **중요한 폴더는 미리 백업**하세요.

## 5. 예제 2 — 웹 크롤링으로 정보 수집

> **사용 환경: claude.ai 웹 또는 Claude Desktop 앱**

### 목표
웹페이지나 검색 결과의 정보를 모아 **표로 정리**하고, 엑셀·문서로 내보냅니다.

### 준비
- **웹 검색(Web search) 기능**을 켭니다. (**설정 → 기능**)

### 진행 단계
1. 웹 검색 기능이 켜져 있는지 확인합니다.
2. 특정 페이지 정리를 요청합니다.
   - "이 페이지(URL)의 핵심 내용을 표로 정리해줘"
3. 검색 결과 수집을 요청합니다.
   - "'생성형 AI 교육' 주제로 최신 기사 5개를 찾아, 제목·요약·링크를 표로 만들어줘"
4. 결과를 파일로 내보냅니다.
   - "위 표를 엑셀 파일로 만들어줘" → 생성된 파일을 다운로드

### 결과 확인
표의 내용과 **출처 링크**가 맞는지 확인합니다.

> ⚠️ 웹 정보는 부정확하거나 오래됐을 수 있으니 **출처를 확인**하세요.
> 로그인이 필요하거나 수집을 금지한 사이트는 가져오지 못할 수 있습니다.

## 6. 예제 3 — 영수증 사진을 엑셀로 정리

> **사용 환경: claude.ai 웹 또는 Claude Desktop 앱**

### 목표
영수증 사진 여러 장을 올려, **날짜·가맹점·품목·금액을 추출**하고 엑셀로 정리합니다.

### 준비
- 별도 커넥터는 필요 없습니다. (이미지 인식 + 파일 생성 스킬을 사용)
- 영수증 사진을 미리 PC에 준비합니다.

### 진행 단계
1. 새 대화를 시작합니다.
2. 영수증 사진들을 대화창에 **끌어다 놓거나**, `+`(첨부) 버튼으로 올립니다.
3. 추출을 요청합니다.
   - "이 영수증들에서 **날짜, 가맹점, 품목, 금액**을 읽어 표로 정리해줘"
4. Claude가 만든 표를 **검토**합니다. (특히 흐릿하게 찍힌 숫자·금액)
5. 엑셀로 내보냅니다.
   - "이 표를 엑셀(`.xlsx`) 파일로 만들어줘" → 파일 다운로드
6. (선택) "월별 합계와 총합계 행도 추가해줘"

### 결과 확인
다운로드한 엑셀 파일을 열어, 금액과 합계가 영수증과 일치하는지 확인합니다.

> ⚠️ 흐릿한 사진은 숫자를 **잘못 읽을 수 있습니다** — 금액·합계는 반드시 검토하세요.
> 영수증의 **카드번호 등 민감정보**가 찍혀 있지 않은지 주의하세요.

## 7. 팀 협업에 활용하기

위 예제로 만든 결과물을 팀원과 함께 쓰는 방법입니다. (claude.ai 팀 플랜 기준)

- **프로젝트(Project)로 묶기**: 관련된 대화·파일을 하나의 Project에 모아 두면,
  팀원이 같은 맥락에서 이어서 작업할 수 있습니다.
- **대화 공유**: 완성된 대화를 **링크로 공유**해 결과를 빠르게 전달합니다.
- **팀 워크스페이스**: 관리자가 멤버를 초대하고, 커넥터를 **팀 공통**으로 설정할 수 있습니다.
- **활용 예**: 예제 3의 "영수증 → 엑셀" 대화를 Project로 만들어 두면,
  경비 담당자가 매달 같은 방식으로 정리를 이어갈 수 있습니다.

## 8. 자주 묻는 점 · 주의사항

### 문제 해결

| 증상 | 해결 |
|---|---|
| 커넥터/기능 메뉴가 안 보임 | 앱·웹을 최신으로 업데이트. 팀 플랜은 **관리자가 해당 기능을 허용**했는지 확인 |
| 파일 커넥터가 폴더를 못 봄 | 커넥터의 **접근 허용 폴더**에 그 폴더가 포함됐는지 확인 |
| 웹 검색이 동작하지 않음 | 설정에서 **웹 검색 기능이 켜져 있는지** 확인 |
| 엑셀 파일이 안 만들어짐 | 파일 생성 기능/스킬이 켜져 있는지 확인 후 다시 요청 |
| 영수증 숫자가 틀림 | 사진을 더 선명하게, 한 번에 **너무 많은 장수**를 올리지 않기 |
| 앱과 웹의 대화가 다름 | 앱과 웹이 **같은 계정**으로 로그인됐는지 확인 |

### 주의사항

1. **결과는 항상 사람이 검토**하세요 — AI는 숫자·사실을 틀릴 수 있습니다.
2. **민감정보 주의** — 주민번호, 카드번호, 영업비밀 등은 업로드 전에 팀의 보안 정책을 확인하세요.
3. **파일 변경은 백업 후** — 폴더를 정리·수정하는 작업은 원본을 백업한 뒤 진행하세요.
4. **권한 확인** — 팀 플랜에서는 일부 커넥터·기능을 관리자만 켤 수 있습니다.
