# VSCode에서 Codex 사용하기 — OpenAI Codex 확장 프로그램

VSCode 안에서 OpenAI의 AI 코딩 도구 **Codex**를 사용하는 방법입니다.

---

## 1. 소개

**Codex**는 OpenAI가 만든 AI 코딩 도구입니다.
VSCode 확장 프로그램으로 설치하면, 편집기 안에서 Codex와 대화하며
코드 작성·수정·설명·오류 수정 같은 일을 맡길 수 있습니다.

---

## 2. 준비물

| 항목 | 설명 |
|---|---|
| VSCode | 설치 완료 (루트 [README](../README.md)의 "개발 환경 준비" 참고) |
| OpenAI 계정 | [chatgpt.com](https://chatgpt.com) 의 ChatGPT 계정. 충분히 쓰려면 **유료 플랜(Plus/Pro)** 또는 OpenAI API 키 권장 |
| 인터넷 연결 | AI 응답은 서버에서 처리되므로 필수 |

---

## 3. 확장 프로그램 설치

1. VSCode 왼쪽 **활동 막대**에서 **확장(Extensions)** 아이콘을 클릭합니다.
   (단축키: `Ctrl + Shift + X`)
2. 검색창에 **`Codex`** 를 입력합니다.
3. 게시자(publisher)가 **OpenAI** 인 항목이 맞는지 확인한 뒤 **Install** 버튼을 누릅니다.
4. 설치가 끝나면 왼쪽 활동 막대에 **Codex 아이콘**이 생깁니다.

> 💡 이름이 비슷한 확장이 여러 개 있습니다. 반드시 **게시자가 OpenAI**인지 확인하세요.

---

## 4. 로그인 (계정 연결)

1. 왼쪽 활동 막대의 **Codex 아이콘**을 클릭해 패널을 엽니다.
2. **`Sign in`**(또는 `Sign in with ChatGPT`) 버튼을 누르면 브라우저가 열립니다.
3. 브라우저에서 OpenAI(ChatGPT) 계정으로 로그인하고 **권한을 허용**합니다.
4. VSCode로 돌아오면 로그인 완료 — 채팅 입력창이 활성화됩니다.

> API 키로 연결하려면, 패널 설정에서 **API Key** 항목에
> [platform.openai.com](https://platform.openai.com) 에서 발급한 키를 넣습니다.

---

## 5. 사용 방법

### 기본 대화
- Codex 패널 아래쪽 입력창에 질문이나 요청을 한국어로 입력합니다.
  - 예) "이 함수가 하는 일을 설명해줘", "로그인 오류를 고쳐줘"

### 코드에 대해 질문하기
- 편집기에서 코드를 **드래그로 선택**한 뒤 Codex에게 물어보면, 그 코드를 기준으로 답합니다.

### 파일 수정 맡기기
- "○○ 기능을 추가해줘" 처럼 요청하면 Codex가 **수정안을 제안**합니다.
- 제안된 변경 내용을 **검토한 뒤 수락(Accept)** 하면 실제 파일에 반영됩니다.

### 터미널에서 쓰기 (선택)
- VSCode 터미널에서 `codex` 명령으로도 같은 도구를 사용할 수 있습니다.

---

## 6. 자주 묻는 점 / 주의

| 상황 | 해결 |
|---|---|
| 검색해도 확장이 안 보임 | VSCode를 최신 버전으로 업데이트 후 다시 검색 |
| 비슷한 확장이 많아 헷갈림 | 게시자가 **OpenAI**인지 반드시 확인 |
| 로그인 창이 안 열림 | 기본 브라우저에서 [chatgpt.com](https://chatgpt.com) 로그인 상태 확인 후 재시도 |
| 응답이 안 옴 | 인터넷 연결, 그리고 플랜의 사용량 한도 확인 |
| 코드가 멋대로 바뀜 | 변경은 **수락하기 전까지 반영되지 않음** — 제안을 항상 검토 후 수락 |

> ⚠️ AI가 만든 코드는 항상 **사람이 검토**해야 합니다. 중요한 파일은 Git으로 먼저 커밋해 두면 안전합니다.
